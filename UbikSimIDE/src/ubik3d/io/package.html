Implements how to read and write {@linkplain ubik3d.model.Home homes} and {@linkplain ubik3d.model.UserPreferences user preferences} created in {@linkplain ubik3d.model model} classes of Sweet Home 3D.

{@linkplain ubik3d.io.DefaultUserPreferences Default user preferences} are read from resources and written in {@linkplain ubik3d.io.FileUserPreferences local files and system preferences}.
Homes are read and written to {@linkplain ubik3d.io.HomeFileRecorder files}.
