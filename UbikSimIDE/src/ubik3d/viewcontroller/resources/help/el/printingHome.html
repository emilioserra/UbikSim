Βοήθεια Sweet Home 3D > Δημιουργία οικιών, τοίχων και επίπλων
     Εκτύπωση οικίας
 

Για εκτύπωση οικίας, επιλέξτε Αρχείο > Εκτύπωση....
Αυτό το μενού εμφανίζει το πλαίσιο διαλόγου εκτύπωσης του συστήματός σας που σας επιτρέπει να επιλέξετε τον εκτυπωτή, τις σελίδες για εκτύπωση, την ποιότητα εκτύπωσης...

Από προεπιλογή, το Sweet Home 3D εκτυπώνει τη λίστα επίπλων, το σχέδιο και την τρέχουσα προβολή 3D της οικίας χρησιμοποιώντας το προεπιλεγμένο μέγεθος χαρτιού, τα προεπιλεγμένα περιθώρια και προσανατολισμό.
Αν θέλετε να τροποποιήσετε αυτές τις παραμέτρους, επιλέξτε Αρχείο > Διαμόρφωση σελίδας....

Στον πίνακα διαμόρφωσης σελίδας μπορείτε να αλλάξετε το μέγεθος χαρτιού και τον προσανατολισμό κάνοντας κλικ στο κουμπί Μορφή σελίδας. Επίσης, μπορείτε να επιλέξετε αν θέλετε να εκτυπωθεί η λίστα επίπλων, το σχέδιο και η προβολή 3D της οικίας. Αν δεν θέλετε να χρησιμοποιήσετε την κλίμακα σχεδίου που υπολογίζεται αυτόματα για να προσαρμόζεται στη σελίδα κατά τον καλύτερο δυνατό τρόπο, μπορείτε να επιλέξετε άλλη κλίμακα στο πεδίο Κλίμακα.
Μπορείτε επίσης να επιλέξετε την κεφαλίδα ή/και το υποσέλιδο της επιλογής σας τα οποία θα εμφανίζονται σε κάθε σελίδα εκτύπωσης. Τα πεδία κειμένου Κεφαλίδα και Υποσέλιδο δέχονται κείμενα ελεύθερης μορφής καθώς και ορισμένες μεταβλητές που θα αντικατασταθούν από την τιμή τους κατά τη στιγμή της εκτύπωσης. Παρακάτω μπορείτε να βρείτε τις 7 μεταβλητές που διατίθενται κατά τη στιγμή αυτή:

  • $pageNumber για αριθμό σελίδας,
  • $pageCount για συνολικό αριθμό σελίδων,
  • $planScale για την κλίμακα του σχεδίου (είτε αυτήν που υπολογίζεται αυτόματα είτε αυτήν που εισαγάγατε στο πεδίο Κλίμακα),
  • $date για την ημερομηνία κατά την έναρξη εκτύπωσης,
  • $time για την ώρα κατά την έναρξη εκτύπωσης,
  • $name για το όνομα της οικίας σας,
  • $file για τη διαδρομή αρχείου του αρχείου της οικίας σας.

Για να αποφύγετε την πληκτρολόγηση του ακριβούς ονόματος της μεταβλητής, χρησιμοποιήστε τα κουμπιά Μεταβλητές που εμφανίζονται στα πεδία κειμένου Κεφαλίδα και Υποσέλιδο . Καθώς το σύμβολο $ διατηρείται για τις μεταβλητές, πρέπει να χρησιμοποιήσετε τον κωδικό $$ για να εκτυπώσετε το σύμβολο $.

Πριν εκτυπώσετε την οικία σας μπορείτε να κάνετε προεπισκόπηση της διαμόρφωσης σελίδας σας στην οθόνη επιλέγοντας Αρχείο> Προεπισκόπηση εκτύπωσης... από το μενού.

Στον πίνακα προεπισκόπησης εκτύπωσης μπορείτε να δείτε πώς θα εκτυπωθεί η οικία σελίδα προς σελίδα. Για να αλλάξετε τη σελίδα προεπισκόπησης κάντε κλικ στα βέλη στο πάνω μέρος του πίνακα ή πατήστε τα πλήκτρα βέλους κατεύθυνσης.

•

Οι ιδιότητες εκτύπωσης μπορεί να μην λειτουργούν στο λειτουργικό σύστημα Linux. Σε αυτή την περίπτωση παρακαλώ επιλέξτε Αρχείο > Εκτύπωση σε PDF... και εκτυπώστε το αρχείο PDF που θα δημιουργηθεί με τα εργαλεία του συστήματός σας.

•

Η εκτύπωση προβολής 3D μπορεί να μην διατίθεται λόγω περιορισμένης διαμόρφωσης του συστήματός σας. Σε αυτή την περίπτωση το κουτί επιλογής Εκτύπωση προβολής 3D θα αποεπιλεχθεί και θα απενεργοποιηθεί.

•

Η προβολή 3D της οικίας εκτυπώνεται πάντοτε σε ανάλυση 150 dpi και κατά τρόπο ώστε να γεμίζει μία σελίδα εκτός των περιθωρίων του χαρτιού.

•

Μπορείτε επίσης να εμφανίζετε την κλίμακα του σχεδίου στο χαρτί σχεδιάζοντας στο σχέδιο διάσταση που να μετρά τυπικό μήκος (1 μέτρο για παράδειγμα).

•

Η διαμόρφωση σελίδας που επιλέγετε αποθηκεύεται με το αρχείο της οικίας.

•

Ο πίνακας προεπισκόπησης εκτύπωσης μπορεί να μεγεθυνθεί.
