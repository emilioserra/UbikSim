Sweet Home 3D Hilfe > Erstellen von Wohnungen, Wänden und Möbeln
     Erstellen einer Wohnung
 

Um eine Wohnung zu erstellen, benutzen Sie einfach die Default-Wohnung, die beim Start von Sweet Home 3D geladen wird, oder klicken auf Datei > Neu.
Wurde eine Wohnung erstellt, können Sie die deren Wände einzeichnen, Türen, Fenster und Möbel hinzufügen, Räume anhand der gezeichneten Wände definieren, die geografische Position anpassen, Bemaßungen einzeichnen und Texte hinzufügen. Da die 3D-Ansicht immer synchronisiert wird, zögern Sie nicht und schauen Sie sich die Änderungen jederzeit in 3D an.

Sie können so viele Wohnungen gleichzeitig editieren wie Sie wollen, aber Sie sollten sich darüber bewusst sein, dass Ihr verfügbarer Arbeitsspeicher eine Grenze darstellt.

•

Die Standard-Höhe von Wänden in einer neuen Wohnung wird von dem Wert abgeleitet, den Sie in den Einstellungen für die Höhe neuer Wände einstellen können. Nach der Erstellung kann jede Wand eine eigene Höhe erhalten, indem Sie diese editieren.

•

Die Standardgröße des Bodens in einer neuen Wohnung ist 10x10 Meter. Wenn eine Wand oder ein Möbelstück außerhalb platziert wird, wird der Plan automatisch vergrößert.

•

Wenn die Luftansicht ausgewählt ist, dann bleibt die 3D-Ansicht immer auf das Wand- und Möbelset zentriert.
