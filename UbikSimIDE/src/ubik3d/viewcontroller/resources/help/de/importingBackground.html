     Importieren eines Hintergrundbildes
 

Um ein Bild zu importieren, welches als Hintergrund im Plan angezeigt werden soll um den Zeichnungsprozess zu beschleunigen, wählen Sie Plan > Hintergrundbild importieren....
Dieses Menü zeigt einen Assistenten der Ihnen helfen soll, ein Bild in wenigen Schritten auszuwählen und zu skalieren.

1.

Klicken Sie auf Bild auswählen und wählen Sie ein BMP, JPEG, GIF oder PNG Bild.

2.

Nachdem das Bild geladen ist, klicken Sie auf Weiter.

3.

Definieren Sie die Skala des Bilder indem Sie die farbige Linie im Bild verschieben. Versuchen Sie, diese so genau wie möglich zu positionieren bis sie eine bekannte Länge exakt überdeckt. Dann schreiben Sie die reale Länge dieser Linie in das Feld und klicken auf Weiter.

4.

Definieren Sie den Ursprung des Bildes im Plan, also der Punkt im Bild, welcher den Koordinaten (0, 0) im Plan gleicht. Klicken Sie nun auf Fertig.

Nachdem der Assistent geschlossen ist, erscheint Ihr Bild im Hintergrund des Planungsfenster. Sie können nun beginnen, Wände zu zeichnen.

und setzen von Türen und Fenstern

•

Um die Skala des Hintergrundbildes präzise zu definieren, vergrößern Sie dieses im Assistenten.

•

Vermeiden Sie das importieren von zu großen Dateien.

•

Wenn Sie ein eingescanntes Bild des Planes Ihrer Wohnung importieren, stellen Sie sicher, dass das Bild korrekt ausgerichtet ist. Eine Rotation von nur einem Grad lässt Sie die Ausrichtung von Grenzen und Wänden verlieren.

•

Die Maßeinheit, welche standardmäßig dargestellt wird, hängt von Ihrem Land ab. Sie können diese natürlich in den Einstellungen verändern.
