     Dimensionen/Bemaßungen zeichnen
 

Um eine Dimension/Bemaßung einzuzeichnen, müssten Sie zuerst im Menü Plan > Bemaßung erstellen oder das Zeichnet Bemaßungen ein -Werkzeug auswählen.

  Zeichnet Bemaßungen ein -Werkzeug

Sie können neue Dimensionierungen im Plan auf verschiedene Wege erstellen:

  • Klicken Sie auf den Startpunkt einer neuen Dimension, klicken Sie auf den Endpunkt, und klicken Sie ein drittes Mal, nachdem Sie den Mauszeiger bewegt haben, um die Länge der Erweiterungslinien zu wählen, die am Ende jeder Dimensionslinie gezeichnet wird
  • Bewegen Sie den Mauszeiger auf ein Ende eines Möbelstücks, einer Wand oder eines Raumes, dem Sie eine Bemaßung/Dimension geben wollen, klicken Sie doppelt, um die temporäre Bemaßung zu akzeptieren, dann klicken Sie ein drittes Mal, nachdem Sie die Länge der Erweiterungslinien ausgewählt haben.

In beiden Fällen haben die neuen Bemaßungen keine Erweiterungslinien, wenn Sie die Maus nicht zwischen dem zweiten und dritten Klick bewegt haben.
Drücken Sie jederzeit die Escape-Taste, um die Erstellung einer Bemaßung/Dimension abzubrechen.
Während Sie den Endpunkt einer Dimension wählen, wird jede Mausbewegung ein Update des Planes bzw. der angezeigten Größe und Länge der Dimension hervorrufen.

Um das Zeichnen von Dimensionen/Bemaßungen zu beenden, wählen Sie Plan > Auswählen oder wählen Sie das Auswahl -Werkzeug an.

 Auswahl -Werkzeug

•

Um Ihnen zu helfen, eine Bemaßung/Dimension präzise einzuzeichnen, benutzen Sie bitte die "Ausrichtungslinien", die von der aktuellen Mauszeiger-Position zur nächsten Wandecke, Raumende oder Möbelecke angezeigt werden. Wenn Sie die Enter-Taste drücken, während der Plan den Fokus hat, wird ein editierbarer Tooltip angezeigt, mit dem Sie per Tastatur die Länge und den Winkel einer Dimension angeben können (Haben Sie noch keine neue Dimension erstellt, wird der Tooltip die Koordinaten des Startpunktes für eine neue Dimension anzeigen), danach den Abstand, die Länger der Erweiterungslinien etc. Benutzen Sie die Pfeiltasten hoch und runter bzw. die Tab-Taste, um einen anderen Wert zu bearbeiten und bestätigen Sie Ihre Werte mit der Enter-Taste. Wollen Sie die Maus wieder benutzen um eine weitere Dimension zu zeichnen, bewegen Sie sie einfach wieder über den Plan.

•

Um Bemaßungen präzise einzuzeichnen, benutzen Sie die Ausrichtungslinien, die als Hilfe von der Position des Mauszeigers zur nächstgelegenen Wand oder Möbelstück angezeigt werden.

•

Die Position und Ausrichtung der Längenangabe einer Bemaßungslinie hängt von der relativen Position der Start- und Endpunkte ab, und von der Position der Erweiterungslinien.

•

Um parallele Bemaßungen/Dimensionen einzuzeichnen, ist der Winkel standardmäßig auf ein Vielfaches von 15° eingestellt. Sie können dies kurzfristig ändern, indem Sie während des Einzeichnens die Shift-Taste gedrückt halten, oder indem Sie den sogenannten Magnetismus in den Einstellungen deaktivieren.

•

Ist eine Bemaßung einmal erstellt, können Sie jederzeit deren Position und Größe mit der Maus ändern.

•

Die Länge, die über der Bemaßung angezeigt wird, ist standardmäßig auf eine gültige Maßeinheit in Ihrem Land eingestellt. Sie können jedoch in den Einstellungen jederzeit zwischen Zentimeter, Meter, Millimeter und Inch wechseln.

•

Die Höhe der Schriftart, welche benutzt wird um den Text der Bemaßung anzuzeigen, kann geändert werden, indem Sie die Skalierung des Plans ändern. Dies können Sie über Plan > Vergrößern oder Plan > Verkleinern bewerkstelligen.

•

Während das Bemaßung erstellen -Werkzeug ausgewählt ist, sind viele andere Werkzeuge und Menüs deaktiviert. Sollten Sie eines dieser Werkzeuge benötigen, wählen Sie das Auswahl -Werkzeug wieder an.
