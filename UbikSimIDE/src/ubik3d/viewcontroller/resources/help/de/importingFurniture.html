     Importieren von Mobiliar
 

Um ein existierendes 3D-Modell in das Mobiliar und/oder den Plan zu importieren, wählen Sie Mobiliar > Mobiliar importieren... oder ziehen Sie die Datei eines 3D-Modelles per Drag&Drop auf ein Sweet Home 3D-Fesnter.
Diese Aktionen zeigen alle einen Schritt-für-Schritt-Assistenten der Ihnen helfen wird, eine Datei eines 3D-Modelles zu wählen und, falls notwendig, dessen Einstellungen anzupassen.

1.

Klicken Sie auf Modell wählen und wählen Sie eine OBJ, DAE, 3DS, LWS Datei oder eine Zip-Datei die einen der vorgenannten Dateitypen enthält. Wenn das Modell geladen ist, klicken Sie auf weiter.

2.

Richten Sie das Modell mit den Pfeil-Knöpfen aus so dass die Vorderansicht die Vorderseite des Modells anzeigt. Manchmal werden 3D-Modelle inkorrekt gerendert da ihre Rück- und Front- Seiten invertiert sind. Ist dies der Fall, klicken Sie auf die Vorder und Rückseite tauschen Checkbox.
Dann klicken Sie auf Weiter.

3.

Ändern Sie, falls notwendig, den Namen, die Größe, Höhe über dem Boden, Farbe und ob es beweglich ist wie Beispielsweise eine Tür oder ein Fenster.
Wenn ein 3D-Modell in den Plan oder die Wohnungsmöbelliste importiert wurde, können Sie ebenfalls die Zum Katalog hinzufügen Checkbox anklicken.
Wählen Sie die Proportionen beibehalten Checkbox ab falls Sie die Höhe, Tiefe oder Breite unabhängig voneinander verändern möchten.
Klicken Sie auf den Farbe zurücksetzen Knopf wenn Sie die Standardfarben bevorzugen, nachdem Sie bereits die Farbe des Modells geändert haben.
Danach klicken Sie auf Weiter.

4.

Drehen Sie das Modell mit der Maus um den besten Ansichtspunkt des Modells für das Modell-Bild zu finden, dass im Katalog, der Liste und im Plan angezeigt wird. Danach klicken Sie auf Fertig.

Nachdem der Assisten geschlossen ist, tauchen das importierte Modell im Mobiliar und der Liste auf, jenachdem, welche Optionen Sie gewählt haben.
Sie können es wie jedes andere Möbelstück auch bearbeiten
Der Name der importieren Möbel erscheint im Katalog in kursiv. Um die Eigenschaften eines importierten Modells zu bearbeiten, klicken Sie doppelt auf es oder wählen es über das Mobiliar > ändern... Menü aus.

•

Das Importieren von 3D-Modellen per Drag&Drop könnte unter Linux unter Umständen nicht funktionieren.

•

Das Katalog-Mobiliar unterstützt keine Homonyme in einer Kategorie. So lange der Name eines importieren 3D-Modelles nicht einmalig in seiner Kategorie vorkommt, bleibt der Weiter Knopf im dritten Schritt des Assistenten deaktiviert.

•

Das Textfeld der Kategorie Combo-Box ist veränderbar für den Fall, dass Sie Ihre eigene Kategorie von Möbeln erstellen möchten.

•

Das Importieren eines 3D-Modelles in das Mobiliar oder in die Wohnung hängt davon ab, welches der beiden Bereiche den Fokus hat während Sie Mobiliar > Mobiliar importieren... klicken. Falls die Möbelliste, der Plan oder die 3D-Ansicht den Fokus hat wenn Sie dieses Menü auswählen, erscheint das importierte Modell nur im Katalog, wenn Sie die entsprechende Checkbox ausgewählt haben im dritten Schritt des Assistenten.

•

Die Maßeinheit, welche standardmäßig angezeigt wird im Assistenten hängt von Ihrem Land ab. Sie können zu Zentimeter, Meter, Millimeter oder Inch wechseln in den Einstellungen.

•

Da der Katalog von allen Sweet Home 3D-Fenstern geteilt wird, können keine Veränderungen an diesem Katalog rückgängig gemacht werden. (z.B. Importe, Modifizierungen, Löschungen).

•

Die Einstellungen von importiertem Mobiliar werden automatisch in deren Einstellungen, 3D-Modell und Icon-Dateien von Sweet Home 3D gespeichert.
