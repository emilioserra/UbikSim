Sweet Home 3D Hilfe > Erstellen von Wohnungen, Wänden und Möbeln
     Speichern und öffnen einer Wohnung
 

Um eine Wohnung zu speichern, wählen Sie Datei > Speichern or Datei > Speichern unter....
Jede Sweet Home 3D-Datei hat die .sh3d-Dateinamenserweiterung. Wenn Sie diese Erweiterung nicht eintippen, wird Sweet Home 3D diese automatisch zu ihrem gewählten Dateinamen hinzufügen.
Nachdem die Wohnung gespeichert ist, erscheint der Dateiname in der Titelleiste von Sweet Home 3D und unter dem Datei > Letzte Häuser öffnen Menü.

Sie können außerdem:

  • den Plan im SVG (Scalable Vector Graphics)-Format exportieren, indem Sie Plan > Exportieren in das SVG Format... auswählen,
  • ein Bild der 3D-Ansicht im PNG-Format erstellen, in dem Sie 3D-Ansicht > Foto erstellen... auswählen,
  • ein Video im Quicktime MOV-Format erstellen anhand eines Pfades in der 3D-Ansicht, indem Sie 3D-Ansicht > Video erstellen... auswählen,
  • die 3D-Ansicht in das OBJ-Format exportieren, indem Sie 3D-Ansicht > Exportieren in das OBJ Format... auswählen. Dieses Feature lässt Sie die OBJ-Datei auswählen, die alle Objekte in der 3D-Ansicht beschreibt, zusammen mit der MTL-Datei und den Textur-Bildern, die alle zusammen im gleichen Ordner angelegt werden.

Um ein existierendes Haus zu öffnen, klicken Sie doppelt auf die Datei, wählen Datei > Öffnen... oder wählen die Datei im Datei > Letzte Häuser öffnen Menü.

•

Eine Sweet Home 3D-Datei speichert den plan einer Wohnung, die Hintergrundbilder die Sie importiert haben, die 3D-Ansichtseinstellungen, und alle 3D-Modelle von Möbeln, die Sie hinzugefügt haben. Dieses Feature erlaubt Ihnen den Austausch von Dateien mit Benutzern, die nicht denn gleichen Möbelkatalog besitzen wie Sie.

•

Wählen Sie Datei -> Speichern und komprimieren, um Ihre Wohnung in einem komprimierten Format abzulegen, welches voll kompatibel mit Sweet Home 3D ist. Diese Aktion wird etwas länger benötigen als das einfache speichern, erzeugt aber wesentlich kleinere Dateien.

•

Wählen Sie Datei > Letzte Häuser öffnen > Verlauf löschen wenn Sie die Liste der letzten Häuser löschen wollen.
