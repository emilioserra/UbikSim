     Importation de meubles
 

Pour importer un modèle 3D existant dans le catalogue des meubles et/ou dans le plan du logement, choisissez le menu Meubles > Importer meuble... ou glissez-déposez le fichier d'un modèle 3D dans une fenêtre de Sweet Home 3D.
Ces opérations affichent un assistant qui vous aide en quelques étapes à choisir le fichier d'un modèle 3D si nécessaire et définir ses caractéristiques.

1.

Si vous avez lancer l'assistant à l'aide du menu, chosissez un fichier au format OBJ, DAE, 3DS, LWS ou un fichier ZIP contenant un fichier à l'un de ces formats en cliquant sur le bouton Choisir modèle. Une fois que le modèle est chargé, cliquez sur Continuer.

2.

Orientez le modèle avec les boutons flèchés de telle façon que la vue de face affiche la face avant du modèle 3D. Quelques fois le rendu de certains modèles 3D est incorrect parce que leurs faces avant et arrière sont inversées. Si c'est le cas, cochez la boîte à cocher Inverser faces avant et arrière.
Cliquez ensuite sur Continuer.

3.

Modifiez si nécessaire le nom, la taille, l'élévation, la couleur du modèle importé et le fait que ce modèle soit déménageable ou que ce soit une porte ou une fenêtre.
Quand un modèle 3D est importé dans la liste des meubles ou le plan du logement, cochez la boîte à cocher Ajouter au catalogue si vous désirez l'importer aussi dans le catalogue des meubles.
Décochez la boîte à cocher Garder les proportions si vous voulez modifier la largeur, la profondeur ou la hauteur du modèle 3D indépendamment l'une de l'autre.
Cliquez sur le bouton Effacer couleur si vous préférez garder les couleurs par défaut du modèle 3D, après que vous en ayez changé sa couleur.
Cliquez ensuite sur Continuer.

4.

Tournez le modèle 3D avec la souris pour obtenir le meilleur point de vue sur le modèle pour l'icône du modèle affichée dans le catalogue des meubles, la liste des meubles et le plan du logement.
Cliquez ensuite sur Terminer.

Une fois l'assitant d'importation de meubles fermé, le modèle importé apparaîtra dans le catalogue des meubles et/ou dans le plan et la liste des meubles du logement, selon les options que vous avez choisies.
Vous pouvez modifier ce modèle dans le plan du logement de la même façon que pour un meuble ajouté depuis le catalogue par défaut.
Le nom des meubles importés dans le catalogue apparaissent en italique. Pour y modifier les caractéristiques d'un modèle importé avec l'assistant importation de meubles, double-cliquez sur ce modèle, ou sélectionnez le et choisissez Meubles > Modifier....

•

L'importation d'un modèle 3D à l'aide d'un glisser-déposer de son fichier dans une fenêtre Sweet Home 3D peut ne pas fonctionner sous Linux.

•

Il n'est pas possible que les meubles d'une même catégorie du catalogue soient homonymes. Tant que le nom du modèle 3D importé dans le catalogue n'est pas unique dans sa catégorie, le bouton Continuer sera désactivé à la troisième étape de l'assistant importation de meubles.

•

Vous pouvez créer une nouvelle catégorie de meubles en saisissant le nom de votre choix dans le champ de saisie de la liste déroulante Catégorie.

•

L'importation d'un modèle 3D dans le catalogue des meubles ou dans le logement dépend du fait que le catalogue ait le focus ou non au moment où vous choisissez Meubles > Importer meuble.... Si la liste des meubles, le plan du logement ou la vue 3D a le focus au moment où vous choisissez ce menu, le modèle importé apparaîtra dans le catalogue uniquement si vous avez coché la boîte à cocher Ajouter au catalogue, à la troisième étape de l'assistant importation de meubles.

•

L'unité affichée par défaut dans l'assistant importation de meubles dépend de votre pays. Vous pouvez choisir entre le Centimètre, le Mètre, le Millimètre et le Pouce dans le panneau des préférences.

•

Comme le catalogue des meubles est partagé par toutes les fenêtres Sweet Home 3D, aucune des opérations effectuées sur le catalogue (c'est-à-dire importation, modifications et suppression) ne peut être annulée.

•

Les caractéristiques des meubles importés du catalogue sont automatiquement enregistrées dans les préférences, et leur modèle 3D ainsi que leur fichier d'icône sont enregistrés dans des dossiers spécifiques de Sweet Home 3D.
