Sweet Home 3D Help
  Подсказки
 

Здесь несколько подсказок, помогающих сделать работу в Sweet Home 3D быстрой и удобной.

Основное

•

Используйте разделители окон между каталогом фурнитуры, списком фурнитуры, планом, видом 3D для изменения размера окон под Ваши предпочтения.

•

Для того, чтобы использовать все операции без риска, помните, что есть операция отката, которая работает везде, кроме изменений в каталоге импорта фурнитур.

Каталог фурнитуры

•

Для добавления более, чем одного объекта на план за раз, выберите нужные объекты в каталоге.

•

Для открытия категории каталога, дважды кликните на ней.

•

Для изменения импортированной фурнитуры, дважды кликните на ней.

•

Для работы с большим каталогом мебели и фурнитуры поможет настройка Список поиска. Окно выбора категорий и текстовая строка поиска помогут найти необходимые атрибуты.

Список фурнитуры дома

•

Для сортировки списка фурнитуры, кликните на заголовок колонки по которой хотите сортировать. При первом клике будет сортировка по возрастанию, при повторном - по убыванию, при третьем клике реализуется сортировка по порядку добавления в список.

•

Можете изменить порядок колонок, просто перетаскивая заголовки в нужном Вам порядке.

•

Для поиска места на плане, где расположена фурнитура, выделите ее в списке фурнитур дома.

•

Для изменения фурнитуры, кликните на ней дважды в плане.

•

Для изменения более чем одной фурнитуры, выделите их на плане или списке, и выберите меню Фурнитура > Изменить....

•

Если возникает необходимость объединить предметы, отметьте их и сгруппируйте Фурнитура > Группа. Такая группа предметов может быть перемещена, изменена или вращаться, как единый элемент.

План дома

•

Если у Вас есть отсканированный план дома, Вы можете его "подложить" под план с помощью импорта фонового рисунка в план перед работой. Это поможет Вам быстрее и точнее нарисовать стены.

•

Для точных корректировок используйте масштабирование.

•

Если хотите сделать стены разноцветными, то составьте их из отдельных сегментов с помощью создания.

•

Для создания стен с наклоном, выберите различную высоту в начале и конце стены.

•

При выборе Редактировать > Копировать или Редактировать > Вырезать, выбранные объекты в плане будут скопированы в клипборд как картинки, и Вы сможете вставить их в другие приложения. Под Linux это не работает. Если хотите сохранить план в векторном формате, выберите План > Экспорт в формате SVG... выйдете из меню.

•

Используйте стрелки клавиатуры для точного позиционирования фурнитуры.

•

Фиксировать основной план выбрать и устроить мебель, не раздражаясь другими видами пунктов, показанных в плане.

•

Используйте выравнивающие линии для точного расположения объектов.

•

Магнетизм, настроенный в панели настроек, может быть временно изменен, если при операции зажать кнопку Shift.

•

В помощь для расположения фурнитуры на плане можно изменить точку просмотра и прозрачность стен в виде 3D.

•

Для перемещения видимой части плана без использования полос прокрутки справа и внизу окна, можно воспользоваться пунктом меню План > Перемещение. Другой вариант активации этого способа перемещения может быть осуществлен нажатием и удерживанием клавиши пробел.

•

Изменение масштаба изображения возможно с помощью пунктов меню План > Увеличить или План > Уменьшить. Дальнейшее увеличение или уменьшение масштаба возможно прокруткой колеса мыши вперед или назад с одновременно удерживаемой клавишей (под ОС Mac OS X, нажать cmd).

Home 3D view

•

Выбор меню Вид 3D > Показать в другом окне позволит переместить окно 3D вида в любое место экрана. Чтобы вернуть 3D представление в главное окно достаточно выбрать Вид 3D > Показать в главном окне или закрыть окно 3D.

•

Если хотите видеть фурнитуру в каждой комнате, измените прозрачность стен.

•

Если яркость завышена и все блестит, измените яркость освещения.

•

Если Вы не можете найти положение виртуального визитера, то крутите колесо мышки, пока масштаб не уменьшится.
