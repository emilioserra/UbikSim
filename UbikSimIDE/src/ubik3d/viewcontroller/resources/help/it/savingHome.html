Aiuto di Sweet Home 3D > Creare case, muri e arredamento
     Salvare e aprire una casa
 

Per salvare una casa, scegli File > Salva oppure File > Salva come....
Ogni file di Sweet Home 3D porta l'estensione .sh3d. Se non digiti questa estensione, Sweet Home 3D la aggiungerà automaticamente al nome.
Dopo il salvataggio della casa, il suo nome appare nel titolo della finestra principale e nel menù File > Apri casa recente.

Puoi inoltre:

  • esportare la piantina della casa in un file formato SVG (Scalable Vector Graphics), scegliendo la voce di menù Piano > Esporta in formato SVG,
  • Crea un'immagine in formato PNG della vista 3D, scegliendo la voce di menù Vista 3D > Crea foto...,
  • esportare la vista 3D di una casa in formato OBJ scegliendo la voce di menù Vista 3D > Esporta in formato OBJ.... Questa caratteristica ti consente di scegliere il file OBJ che descriverà tutti gli oggetti della vista 3D, con il suo file MTL e le textures che saranno create nella stessa cartella.

Per aprire una casa esistente, fai doppio click sul suo file, scegli File > Apri... o scegli uno dei file tramite il menù File > Apri casa recente.

•

Un file Sweet Home 3D comprende la piantina di una casa, l'immagine di sfondo che hai importato, le caratteristiche della vista 3D impimpostate, e tutti i modelli 3D dell'arredamento che hai aggiunto a questa casa. Questa caratteristica ti consente di condividere o scambiare il file della tua casa con persone che non hanno lo stesso catalogo di arredamento.

•

Scegli File > Salva e comprimi per salvare la tua casa in un formato compresso pienamente compatibile con il formato di default di Sweet Home 3D. Questa operazione prende più tempo ma creerà un file più piccolo.

•

Scegli File > Apri casa recente > Cancella menu se vuoi cancellare la lista dei file di Sweet Home 3D files che compaiono nel menù File > Apri casa recente.
