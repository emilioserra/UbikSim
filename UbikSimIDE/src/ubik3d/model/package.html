Describes model classes of Sweet Home 3D. This is the core package of Sweet Home 3D upon which all the other packages are dependent.

The {@link ubik3d.model.HomeApplication HomeApplication} abstract class is the entry point of this package, and gives access to {@linkplain ubik3d.model.UserPreferences user preferences}, to the list of created {@linkplain ubik3d.model.Home homes}, with their {@linkplain ubik3d.model.Wall walls}, {@linkplain ubik3d.model.HomePieceOfFurniture furniture},... and how to {@linkplain ubik3d.model.HomeRecorder read and write homes}.
