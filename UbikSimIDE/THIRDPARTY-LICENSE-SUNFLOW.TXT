SunFlow library is used for global illumination rendering.

SunFlow library is available at:
http://sourceforge.net/projects/sunflow/ 
To build sunflow-0.07.3g.jar file, download the source code of its unreleased 0.07.3 version with the following SVN command:
svn co -r 396 https://sunflow.svn.sourceforge.net/svnroot/sunflow/trunk sunflow
and use sunflow-0.07.3g-src-diff.zip included in Sweet Home 3D project for source differences and instructions
